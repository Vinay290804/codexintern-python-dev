
# CodexIntern July 2025 - Python Projects

This repository contains implementations of selected tasks for the CodexIntern Python Development Internship - July 2025.

## Completed Tasks

### 1. CSV Data Analysis & Visualization
- Performed data analysis using Pandas.
- Visualized trends with Matplotlib and Seaborn.
- Insights derived from correlation and distribution patterns.

### 2. House Price Prediction using Linear Regression
- Used Scikit-learn to build a predictive model.
- Dataset included house features like area, rooms, and location.
- Evaluated model performance using Mean Squared Error.

### 3. Flask Web App for Sentiment Analysis
- Developed a web interface with Flask.
- Used TextBlob for natural language sentiment analysis.
- User input is classified as Positive, Negative, or Neutral.

## How to Run

1. Clone the repository.
2. Install the required packages:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the corresponding Python file for each project.

## Requirements

- Python 3.x
- pandas, matplotlib, seaborn
- scikit-learn
- flask
- textblob
